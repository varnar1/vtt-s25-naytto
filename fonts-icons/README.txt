Fonts and Icons Used in This Project
=====================================

This folder contains all the fonts and icons used in the project.

FONTS:
------
Inter Font Family (Google Fonts)
- Inter-Regular.ttf (weight: 400)
- Inter-Medium.ttf (weight: 500)
- Inter-SemiBold.ttf (weight: 600)

ICONS:
------
Material Icons (Google Fonts)
- MaterialIcons-Regular.ttf

CSS FILES:
----------
- inter-fonts.css - Original CSS from Google Fonts for Inter
- material-icons.css - Original CSS from Google Fonts for Material Icons

SOURCE LINKS:
-------------
Inter Font: https://fonts.google.com/specimen/Inter
Material Icons: https://fonts.google.com/icons

All files downloaded from Google Fonts and are free to use.
