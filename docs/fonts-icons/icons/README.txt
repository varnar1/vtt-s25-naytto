Individual Icon Files
=====================

This folder contains individual SVG icon files for each icon used in the project.

ICONS USED IN THE PROJECT:
--------------------------
1. menu.svg - Hamburger menu icon (header)
2. palette.svg - Logo icon (header and copyright)
3. close.svg - Close icon (mobile menu)
4. arrow_forward.svg - Arrow icon (hero search)
5. account_tree.svg - Branches feature icon
6. group.svg - Organizations/teams feature icon
7. vpn_key.svg - Authentication feature icon
8. account_circle.svg - Account management feature icon
9. payment.svg - Billing feature icon
10. chat_bubble_outline.svg - Support feature icon
11. help.svg - Help button icon

All icons are from Google Material Icons and are available as SVG files.
You can open each SVG file in a browser or image viewer to see the icon.

SOURCE:
-------
Material Icons: https://fonts.google.com/icons

Note: The project uses Material Icons as a font (MaterialIcons-Regular.ttf),
but these individual SVG files are provided for reference and can be used
if you want to switch from font-based icons to SVG icons.
